# MemoDemo
由于工作需要,将经常使用的代码整合成一个较大的demo,长期更新...

目前实现:

· 下拉刷新 上拉加载更多

· 使用xUtils3数据库功能

· 使用Choose-Media选择本地视频

· 二级列表

· StepView步骤指示器

· 使用multi-imges-select 选择本地图片  多少自己定

· 加入弹出模块 包括PopupWindow(仿微信)和Activity的弹出动画

· 添加使用节操视频框架播放视频

· 添加ZXing使用案例 包括扫描和生成二维码

· 添加数字日期转为汉字日期的功能

· 添加判断日期为今天或者其他日期的模块
